Лабораторная работа № 4. Организация передачи управляющих команд на удалённое оборудование.

Задание:
Организовать передачу управляющих команд из интерфейса на оборудование для всех вещей.
Передачу управляющих команд можно реализовать как при помощи нескольких запросов (по одному для каждой вещи),
так и одним запросом. В рамках данной работы необходимо реализовать передачу именно управляющих команд;
переделывать запросы мониторинга из ЛР3 не нужно.

Реализовано:
- Маршруты /connect_* — только мониторинг (GET, без команд).
- Маршруты /control_* — управляющие команды (GET с параметрами).
- Метод control() в каждом классе вещи в things.py.
- Кнопки интерфейса отправляют ajax GET-запросы на /control_*.
